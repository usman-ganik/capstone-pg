# Next.js UI Skeleton — Tender Fee Configurator (Modern SaaS)

## Use this with an existing Next.js app

This zip contains two variants:

- `no-src/`  -> copy folders into your project root (recommended if you do NOT have `src/`)
- `with-src/` -> copy the contents of this folder into `src/` (recommended if you DO use `src/`)

## Steps
1) Create your Next.js app (App Router + Tailwind recommended)
2) Add shadcn/ui (optional but recommended):
   - npx shadcn@latest init
   - npx shadcn@latest add button card tabs badge input textarea accordion checkbox
3) Copy files from either `no-src/` or `with-src/` into your project
4) npm install
5) npm run dev

Open http://localhost:3000 and go to /customers
